How to Use RootRecord

  1. Open @rootrecord_bot in Telegram
  2. Send /start
  3. Add vehicle: /vehicle add WZP484 2014 Chevy Cruze 60675
  4. Log fuel: /fillup → enter gallons price [odometer]
  5. Check MPG: /mpg
  6. Track finances: /finance expense 45.67 Gas
  7. Web dashboard: /web in bot → get login link
