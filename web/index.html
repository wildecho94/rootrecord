How to Use RootRecord

  1. Start in Telegram
    Open @rootrecord_bot and send /start
  2. Add a Vehicle
    Use /vehicle add WZP484 2014 Chevy Cruze 60675
  3. Log Fuel
    Send /fillup → enter gallons price odometer → pick vehicle
  4. Check MPG
    Send /mpg to see stats (works after full tanks)
  5. Track Finances
    Use /finance expense 45.67 Gas or /finance balance
  6. Web Dashboard
    Use /web in Telegram → get login link → access full reports
  7. Uptime & Stats
    Send /uptime to see bot reliability

Tips: All data is private and filtered by your Telegram user ID. Live location pings are saved automatically.
